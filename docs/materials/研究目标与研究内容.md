# 研究目标

深度神经网络已广泛应用于图像识别等任务，但模型内部的决策逻辑难以直接观察，对边缘输入的预测稳定性也缺乏充分验证。现有测试方法面临两方面困难。黑盒测试以准确率为核心评价标准，直观但无法触及模型内部的决策结构，对长尾分布下的稀有样本难以系统覆盖。基于神经元覆盖的白盒测试试图打开模型内部，但普通覆盖将所有神经元等权对待，覆盖率的提高并不总是带来缺陷发现能力的同步提升，且生成的测试样本容易偏离有效输入分布。本研究面向图像分类类深度神经网络，在白盒或半白盒条件下（需要访问模型输出概率、中间层激活和输入梯度），研究一种融合差分行为、关键神经元覆盖和语义约束的候选缺陷发现方法。

本研究面向深度神经网络安全测试中"覆盖率不足、测试样本语义缺失及关键失效难以触发"等核心问题，拟构建一种融合差分测试、关键神经元引导与语义约束生成的智能模型安全缺陷发现方法，系统提升测试效率与缺陷发现能力。项目的方法从差分测试出发。框架选用多个结构不同但任务相同的模型互为交叉引用先知，通过共识种子筛选和严格候选缺陷判定，以模型间预测分歧作为候选缺陷信号，降低对人工标注的依赖。在差分信号的基础上，框架深入模型中间层，从大量神经元中筛选与决策关联较强的关键子集，建立关键覆盖和类关键覆盖指标，引导测试向未探索的关键结构方向推进，并利用关键神经元激活模式对缺陷样本进行分类。为了防止生成的样本偏离有效输入分布，框架在变异过程中加入输入语义保持约束，同时利用路径差异反馈增加缺陷样本的多样性。上述目标被纳入一个统一的迭代测试框架，通过动态反馈协调各目标权重，并通过消融实验评价各模块对缺陷发现、覆盖增长、语义质量和缺陷多样性的贡献。

通过上述研究，本研究旨在缓解普通神经元覆盖测试的局限，形成一套兼顾覆盖引导、候选缺陷发现和语义质量控制的图像分类模型安全测试方法。项目的目标不是直接证明所有候选样本都是真实错误，而是提高候选缺陷样本的发现效率，并为后续核验和缺陷分类提供中间层证据。最终效果需要通过消融实验、人工抽检和多维度指标评估验证。

# 研究内容

本研究围绕图像分类模型中的候选误分类、边界样本脆弱性和扰动下预测不稳定等问题，开展基于中间层反馈的安全测试方法研究。研究内容包括四个部分：差分行为驱动的候选缺陷捕获、关键神经元覆盖与缺陷分类、语义约束的测试样本生成、联合优化测试与评估框架。四个部分前后衔接：差分行为提供候选缺陷信号，关键神经元提供结构反馈，语义约束控制样本有效性，联合框架负责组织迭代生成和系统评估。

## 1. 差分行为驱动的缺陷捕获技术

针对深度学习测试中人工标注成本高、长尾样本难以预先获得真实标签的问题，本研究构建基于多模型差分行为的白盒测试机制。框架选择多个面向同一任务、共享输入格式和标签空间但结构不同的模型作为交叉引用 oracle。测试开始前，框架筛选所有模型预测一致的样本作为种子，并将该一致预测记为初始共识标签。变异后，只有当目标模型偏离初始共识，而参考模型仍保持共识时，样本才进入目标模型候选缺陷集合。

在此基础上，本研究构造差分目标函数，引导输入沿目标函数增大的方向变异，使目标模型降低对共识标签的置信度，同时尽量维持参考模型的判断。实验阶段将轮换目标模型，比较不同架构的缺陷触发情况。框架还记录种子接受率、参考共识保持率、候选缺陷触发率和目标置信度变化，用于评估差分模块是否稳定有效。该机制不能发现所有模型共同犯错的情况，候选缺陷仍需结合标签、人工抽检或后续语义检查进行核验。

## 2. 关键神经元覆盖与缺陷分类方法

现有基于神经元覆盖的测试方法通常将所有神经元等价对待，忽略了不同神经元在模型决策过程中的作用差异，导致测试资源分配不均、覆盖效率较低。针对这一问题，本研究从模型中间层结构出发，提出面向关键神经元的覆盖度量方法。

本研究以训练集上的激活频率为基础指标，再结合归因方法估计的决策贡献度作为第二维度，将二者融合为综合关键度指标，筛选出与模型决策关联较强的神经元子集。在此基础上，定义关键神经元覆盖率（CNCov）作为测试反馈指标，引导测试样本优先覆盖与决策更相关的结构。同时，考虑不同类别样本在模型中的决策路径存在差异，进一步构建类别敏感的关键神经元集合和类关键覆盖率（CCCov），揭示类别间的覆盖不均衡，使测试资源向覆盖不足的类别倾斜。

此外，关键神经元的激活模式还可以作为缺陷分类的依据。不同类型的模型缺陷往往对应不同的关键神经元激活组合，因为它们触发了模型内部不同位置的决策异常。通过比较缺陷样本在关键神经元集合上的激活差异，可以将缺陷按其对应的内部决策路径进行分组，从而评估测试所发现缺陷的多样性，避免对同一类缺陷的重复发现。

## 3. 语义约束的测试生成方法

针对覆盖驱动生成样本容易出现语义失真的问题，本研究构建语义约束的测试样本生成机制。该机制区分输入语义保持和决策路径差异两个目标。输入语义保持用于判断变异样本是否仍然保留原始样本的基本视觉内容；决策路径差异用于判断样本是否探索了模型内部不同的表示路径。前者要求中间层特征相似度保持在较高水平，后者鼓励关键神经元激活模式的差异增大，二者不能合并为同一个"相似度越高越好"的指标。

本研究以目标模型中间层特征和关键神经元激活向量作为轻量语义代理，分别计算输入相似度和路径相似度。输入相似度可以作为可微语义项进入联合优化目标，路径相似度则用于候选样本筛选和种子调度。自然语言神经元描述和文本路径相似度可以作为可选的解释路线，但不作为默认优化目标。该机制的效果需要通过输入有效率、样本可视化、人工抽检和路径多样性指标验证。

## 4. 联合优化测试与评估框架

在前三项研究的基础上，本研究构建覆盖驱动与语义引导的联合优化测试与评估框架。框架将差分目标、关键覆盖目标和输入语义保持项组成联合目标，通过 PGD 式梯度更新生成候选样本。路径差异和输入有效性不全部写入可微目标，而是在候选生成后用于外层分流和种子调度。候选样本根据输入有效性、目标模型是否偏离共识、参考模型是否保持共识、是否带来新覆盖和是否探索新路径进行分类处理。

框架在测试过程中记录覆盖增长率、候选缺陷触发率、输入语义偏移和路径新颖样本比例，并据此调整种子优先级和目标权重。动态反馈降低了对静态手工权重的依赖，但初始权重和调整参数仍需通过预实验校准。

为验证框架有效性，本研究设置逐步消融实验，比较纯差分测试、差分加关键覆盖、差分加关键覆盖加语义保持、完整动态反馈四组配置，还需比较静态权重与动态权重的表现差异。实验在 ImageNet 和 CIFAR 类图像分类任务上展开，在 ResNet50、VGG16-BN、MobileNetV2 等异构模型组合上轮换目标模型。评价指标覆盖五个维度：缺陷方面报告候选缺陷数量、RFT 和 FRE，覆盖方面报告 CNCov 和 CCCov 及增长曲线，语义方面报告输入有效率和平均输入相似度，多样性方面报告缺陷簇数量和类别覆盖，效率方面报告运行时间和单位时间缺陷数。

在缺陷多样性评估上，本研究利用关键神经元激活模式对候选缺陷样本进行聚类分析，统计独立缺陷模式的数量和分布特征，判断框架是否在不同决策路径上发现了不同类型的问题，而不只是反复命中同一类缺陷。通过上述指标，本研究评估各模块对缺陷发现、覆盖效率、样本质量和缺陷多样性的实际贡献。
